# Stack Profiles

AgentiUX Dev treats all supported technologies as first-class inputs to routing, stage planning, and verification.

## Supported Technologies

- React
- Next.js
- NestJS
- TypeScript
- React Native
- Expo
- Nativewind
- Tailwind
- Rust
- Nx
- Docker
- Docker Compose
- Docker Swarm
- Postgres
- MongoDB
- Redis
- NATS
- Android
- iOS
- Python
- Codex Plugin
- MCP Server
- Local Dashboard

## Detection Signals

- `package.json`
- `tsconfig.json`
- `nx.json`
- `Cargo.toml`
- `docker-compose*.yml`
- `compose*.yaml`
- `app.json`
- `app.config.*`
- `android/`
- `ios/`
- `Podfile`
- `*.xcodeproj`
- NestJS config files
- service manifests mentioning Postgres, MongoDB, Redis, or NATS
- `*.py` under plugin or script directories
- `.codex-plugin/plugin.json`
- `.mcp.json`
- `dashboard/index.html`
